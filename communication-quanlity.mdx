## **6.1 Sales Team Daily Checklist**

| Task | Notes / Info |
| :-- | :-- |
| ☐ Check all new web form submissions | Ensure no leads are missed |
| ☐ Follow up on pending quotes per schedule | Refer to Follow-Up Processes (Section 2) |
| ☐ Log all customer interactions in CRM | Include date, time, contact method, outcome |
| ☐ Review bookings for tomorrow | Confirm all details are correct |
| ☐ Send confirmation emails for new bookings | Include booking summary & driver info |
| ☐ Update lead statuses | Hot, Warm, Cold, Lost, Converted |

💡 **Info:** Daily review ensures no missed leads and accurate CRM data.

---

## **6.2 Dispatch Team Daily Checklist**

| Task | Notes / Info |
| :-- | :-- |
| ☐ Review all trips for next 24 hours | Check schedule completeness |
| ☐ Assign drivers to all pending trips | Use Driver Assignment Process (Section 3.2) |
| ☐ Send trip sheets to drivers | Include all details, special requests, and route info |
| ☐ Send customer confirmations with chauffeur details | T-24 hours |
| ☐ Check flight statuses for airport pickups | Adjust trip timing if needed |
| ☐ Confirm driver availability and readiness | Follow up with driver as needed |
| ☐ Monitor active trips in real time | Track progress and handle issues |
| ☐ Close completed trips in system | Record actual pickup/drop-off, mileage, notes |
| ☐ Document any incidents | Include any deviations, complaints, or issues |

💡 **Info:** Monitoring ensures smooth operations and quick resolution of issues.

## **6.3 After-Sales Team Daily Checklist**

| Task | Notes / Info |
| :-- | :-- |
| ☐ Send post-ride follow-ups for yesterday's trips | Text \+ Email templates (Section 4.1) |
| ☐ Review customer feedback/survey responses | Identify satisfied and dissatisfied customers |
| ☐ Escalate complaints to management | Follow Complaint Handling Process (Section 4.3) |
| ☐ Process refunds/credits as approved | Ensure proper logging |
| ☐ Request reviews from satisfied customers | Follow Review Request Process (Section 4.2) |
| ☐ Update customer profiles with preferences | Vehicle type, driver, amenities, special requests |
