## **5.1 Phone Etiquette**

| Standard | Details |
| :-- | :-- |
| Answer Time | Within 3 rings |
| Required Info | Ask for name, email, date, time, number of passengers, luggage count, special requests, vehicle preference |
| Tone | Smile when speaking (affects tone) |
| Customer Name | Use at least twice during the call |
| Speech | Avoid filler words (um, uh, like) |
| Interruptions | Never interrupt the customer |
| Call Close | Confirm next steps before ending call |
| Appreciation | Thank the customer for calling |

💡 **Info:** Polished phone etiquette improves professionalism and customer confidence.

---

## **5.2 Email Standards**

| Standard | Details |
| :-- | :-- |
| Response Time | Within 5 minutes during business hours |
| Greeting & Sign-off | Use professional opening and closing |
| Accuracy | Proofread before sending |
| Content | Include all relevant details in the first response |
| Subject Lines | Clear and descriptive |
| Signature | Include contact info |

💡 **Info:** Emails are often the first impression; clarity and professionalism matter.

---

## **5.3 Text Message Guidelines**

| Standard | Details |
| :-- | :-- |
| Length | Keep under 160 characters when possible |
| Tone | Professional but friendly |
| Branding | Include company name |
| Response Time | Within 5 minutes during business hours |
| Abbreviations | Avoid excessive use |
| Callback | Always include callback number |
