---

## **1.1 Hourly Booking Details**

| Field | Notes |
| :-- | :-- |
| Number of Hours | Minimum booking: 4 hours for Bus, Stretch, Sprinter; 3 hours for Sedan and SUV |
| Start Date & Time | Required |
| First Pickup Location | Address for trip start |
| Estimated Stops | Multi-stop input or “Add Stop” button for multiple destinations |
| Final Drop-off Location | Address where the trip ends |
| Area of Service | City only / Outstation? |
| Overtime Rate Acknowledgment | Checkbox to confirm customer understands overtime charges |
