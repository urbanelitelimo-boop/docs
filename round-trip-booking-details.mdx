---
title: "Round trip Booking Details"
---

> Used when a customer requests a return journey. These fields are **in addition** to the standard Customer & Ride Details Form.

---

## **1.1 Return Trip Details**

| Field | Notes |
| :-- | :-- |
| Return Pickup Date | Required for return journey |
| Return Pickup Time | Required for return journey |
| Return Pickup Location | Address where return starts |
| Return Drop-off Location | Address where return ends |
| Waiting Time at Destination | Optional, if customer requires waiting at drop-off location |
